## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> pkg update && pkg upgrade 
> pkg install nodejs
> pkg install git
> pkg install bash
> git clone https://github.com/jepribarus/kangdev
> cd bot
> bash install.sh
```

### Usage
```bash
> node index.js
```
